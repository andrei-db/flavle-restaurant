# Flavle — Luxury Restaurant Landing Page

Flavle is a premium WordPress landing page concept for a luxury fine dining restaurant in Bucharest.

## Features

- Conversion-focused hero section
- Sticky navigation with reservation CTA
- Trust bar and social proof
- Signature dishes section
- Menu preview
- Gallery section
- Testimonials
- FAQ section
- Booking form UI
- Mobile sticky reservation button
- Local SEO schema markup
- FAQ schema markup
- Responsive premium layout

## Built With

- WordPress
- Custom PHP theme
- CSS3
- JavaScript
- Schema.org structured data

## Focus

This project was designed with a strong focus on:

- Premium visual design
- Conversion rate optimization
- Local SEO
- Mobile responsiveness
- Restaurant booking experience
